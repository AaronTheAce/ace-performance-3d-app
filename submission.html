Deeper Understanding Statement

The project not only exceeds the main course content in that it uses Three.js to load and interact with models, swap models, toggle wireframe models, and offers more control options, such as cameras and lights. Moreover, there is a concept of brand identity as well as adaptive user interface design according to varying screen resolutions.

Additional areas that can be explored include integrating the 3D App with PHP and SQLite to capture user feedback from testing, and playing around with custom shaders for creating more stylised visuals of the ACE Performance models.

The website was tested through functional, browser and user testing. Core interactions such as page navigation, model switching, wireframe mode, camera changes, spotlight toggling and spin animation were checked to ensure they operated correctly. Cross-browser testing was completed using Chrome, Edge and Firefox. Device testing was carried out on both laptop and desktop screens to review layout responsiveness and readability. The website was tested on both desktop and laptop screen sizes to ensure layout consistency, readability and correct scaling of the 3D viewer. Peer feedback suggested improving scene brightness, camera framing and button clarity. As a result, lighting intensity was increased, model orientation was corrected and interface spacing was refined.
